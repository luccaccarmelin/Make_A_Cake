--Make a Cake--

Projeto solo Brasileiro feito 100% em Python.

UPDATE 0.3 EM BREVE!

  Como jogar
Baixe o arquivo cake.py.
Abra o Prompt de Comando (CMD)/Terminal.

Vá até a pasta do arquivo usando o comando cd e depois a pasta na onde o arquivo esta (exemplo: cd Documents)

Execute:

py cake.py

ou

python cake.py

ou

python3 cake.py

Para iniciar o jogo.

  -*Requisitos*-

Python 3 instalado: https://www.python.org/downloads/
No instalador do Windows, marque “Add Python to PATH”.

Teste no CMD/Terminal:

python --version

Deve aparecer algo como Python 3.12.x.

-Ferramentas usadas para fazer o jogo: Notepad e CMD/Terminal
